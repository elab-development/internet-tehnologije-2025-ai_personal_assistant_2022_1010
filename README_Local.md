# AI Personal Assistant Platform

University project for the course **Internet Technologies**.

This project implements an AI-powered personal assistant with a web-based interface.
The system is designed using modern web technologies and follows a containerized
architecture for easy deployment and testing.

## Features
- AI-powered assistant logic
- Backend API service
- Frontend web interface
- Dockerized environment
- REST API with documentation

## Tech Stack
- Backend: Node.js / Python (to be finalized)
- Frontend: React
- Containerization: Docker, Docker Compose
- API Documentation: Swagger / OpenAPI
- Version Control: Git & GitHub

## How to Run Locally

```bash
docker-compose up --build
